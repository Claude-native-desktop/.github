# Claude for Mac — Native App From Anthropic With Opus 4.5, Parallel Code Sessions, and Caps Lock Global Invocation

## About Claude for Mac

Claude for Mac provides Anthropic's official native desktop client with Claude Opus 4.5, Haiku, and Sonnet models — accessible globally via Caps Lock shortcut. Claude for Mac combines parallel code sessions (run multiple independent conversations side by side), 200K context window (process entire codebases, novels, transcripts), file uploads (images, PDFs, text, code files), and voice input. Claude for Mac enables developers to debug code in one session, write documentation in another, and answer questions in a third — simultaneously. Claude for Mac includes system-wide Caps Lock invocation (remappable), conversation organization (projects), and artifacts (run JavaScript, Python, or render HTML inline). Claude for Mac is trusted by software engineers, data scientists, technical writers, and researchers who need Claude at their fingertips.

## Deep Dive — Native AI Assistant With Parallel Sessions

Caps Lock global invocation — press Caps Lock to summon overlay from any app, release to dismiss (configurable to other keys)  
Parallel code sessions — tabbed interface, each session independent, switch between debugging, writing, Q&A, analysis  
200K context window — process 200,000 tokens (~150,000 words, or 500+ pages) in single conversation  
Artifacts — Claude generates and runs code snippets, renders HTML/CSS, displays charts, interactive widgets  
Projects — organize conversations by project, attach project-specific files, share project context across sessions  
Native SwiftUI — built for Apple Silicon, minimal resource usage, supports VoiceOver and keyboard navigation

## Quick Reference for Claude for Mac

| Operation | Claude for Mac Action |
|-----------|----------------------|
| Open Claude overlay (global) | Caps Lock (configurable) |
| New conversation (current session) | Cmd+N |
| New parallel session (new tab) | Cmd+T |
| Switch between sessions | Cmd+Tab or Cmd+Shift+Tab |
| Close current session | Cmd+W |
| Upload file | Cmd+Shift+U |
| Run artifact (code) | Click Run button in artifact |
| Copy last response | Cmd+Shift+C |
| Open projects | Cmd+Shift+P |
| Open preferences | Cmd+, |
| Quit Claude | Cmd+Q |
